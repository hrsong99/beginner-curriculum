# Variation E: Simplified Hybrid

## Philosophy

This variation is a **streamlined version of the Full Hybrid (D)**, cutting the vocab page
and simplifying tutor-facing elements to let the lesson flow faster. It keeps the progressive
drill framework and CLT roleplay but trims overhead so tutors spend less time reading
instructions and more time interacting with the student.

Best for: When you want a complete lesson experience in under 20 minutes with less tutor prep.

**Key Features:**
- Clean pattern intro without tutor script (Variation A style)
- Progressive pattern drills: read → fill → translate (John)
- 1 CLT activity with bilingual goal boxes (Scott)
- Korean translations throughout (all)
- "Mission Complete" wrap-up with big checkboxes (Yeji)
- Clear student/tutor separation on every page
- ~20 minutes total

---

## Lesson Flow (9 Pages)

| Page | Activity                     | Time  | Source Inspiration     |
|------|------------------------------|-------|------------------------|
| 1    | Title & Pattern Intro        | 2 min | Variation A            |
| 2    | Guided Conversation          | 3 min | John                   |
| 3    | Memory Conversation          | 3 min | John                   |
| 4    | Pattern Drill A              | 3 min | John                   |
| 5    | Pattern Drill B              | 3 min | John                   |
| 6    | Short Dialogues              | 3 min | John                   |
| 7    | CLT Activity (Roleplay)      | 4 min | Scott                  |
| 8    | Free Discussion              | 3 min | Variation A            |
| 9    | Mission Complete             | 2 min | Yeji                   |

## Sample Lesson: "Be Verb — Introductions & Occupations"

### PAGE 1 — Title & Pattern Intro

```
+-------------------------------------------------------------------------+
|                                                                         |
|  LESSON 1: NICE TO MEET YOU!                                            |
|  ----------------------------                                           |
|                                                                         |
|  Outcome:                                                               |
|  You can introduce yourself and describe people                         |
|  자기소개하고 사람을 설명할 수 있어요.                                           |
|                                                                         |
|                                                                         |
|  TODAY'S PATTERNS                                                       |
|  ================                                                       |
|                                                                         |
|  1. Identity (신원/정체)                                                   |
|                                                                         |
|     * I am a [student].          저는 [학생]입니다.                         |
|     * You are a [teacher].       당신은 [선생님]입니다.                      |
|     * He/She is a [doctor].      그/그녀는 [의사]입니다.                     |
|                                                                         |
|                                                                         |
|  2. Description (묘사)                                                    |
|                                                                         |
|     * I am [happy].              저는 [행복]합니다.                         |
|     * They are [tired].          그들은 [피곤]합니다.                        |
|     * She is [from Korea].       그녀는 [한국] 출신입니다.                    |
|                                                                         |
|                                                                         |
|  [Tutor: Read each pattern aloud. Student repeats after you.]           |
|                                                                         |
|                                                                  PAGE 1 |
+-------------------------------------------------------------------------+
```

### PAGE 7 — CLT Roleplay Activity

```
+-------------------------------------------------------------------------+
|                                                                         |
|  REAL-LIFE PRACTICE: ROLEPLAY 실전 연습                                   |
|  ────────────────────────────────────                                   |
|  "The Welcome Party"                                                    |
|                                                                         |
|  SCENARIO:                                                              |
|  You are at a welcome party for new international students.             |
|  Meet the tutor (another student) and make a new friend!               |
|  상황: 외국인 학생 환영 파티에 왔습니다. 튜터(다른 학생)를 만나세요!              |
|                                                                         |
|                                                                         |
|  ┌─── FOR TUTOR ──────────────────────────────────────────┐            |
|  │                                                         │            |
|  │  GOAL: Help the student have a real conversation.       │            |
|  │                                                         │            |
|  │  RULES:                                                 │            |
|  │  ✓ Start with: "Hi! Are you new here?"                  │            |
|  │  ✓ Use simple sentences. Stay in character.             │            |
|  │  ✗ Don't correct grammar mid-conversation.              │            |
|  │                                                         │            |
|  │  PHRASES:                                               │            |
|  │  * "Are you from ___?"                                  │            |
|  │  * "Are you a ___?"                                     │            |
|  │                                                         │            |
|  └─────────────────────────────────────────────────────────┘            |
|                                                                         |
|                                                                         |
|  ┌─── 학생용 (FOR STUDENT) ────────────────────────────────┐            |
|  │                                                         │            |
|  │  목표: 새 친구를 사귀세요!                                  │            |
|  │                                                         │            |
|  │  규칙:                                                   │            |
|  │  ✓ 자기소개를 하세요 (이름 + 직업)                           │            |
|  │  ✓ "I am..." 문장을 2번 이상 사용하세요                     │            |
|  │  ✗ "Yes" / "No"만 말하지 마세요!                           │            |
|  │                                                         │            |
|  │  이 표현을 사용하세요:                                      │            |
|  │  * "I am [이름]. I am a [직업]."                          │            |
|  │  * "I am from [나라]."                                   │            |
|  │                                                         │            |
|  └─────────────────────────────────────────────────────────┘            |
|                                                                         |
|                                                                  PAGE 7 |
+-------------------------------------------------------------------------+
```
