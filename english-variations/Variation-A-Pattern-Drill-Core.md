# Variation A: Pattern-Drill Core

## Philosophy

This variation focuses on **pattern mastery through progressive repetition**. Inspired
by John's 2-pattern structure and Andrew's visual emphasis, it moves students through
increasing difficulty: guided reading → fill-in → translation → free production.

Best for: Students who benefit from structured repetition and clear scaffolding.

**Key Features:**
- 2 patterns per lesson (John)
- Visual cues / pictures where possible (Andrew)
- Korean translations throughout (all)
- Progressive difficulty within each activity
- 25 minutes total

---

## Lesson Flow (7 Pages)

| Page | Activity                  | Time  | Difficulty  |
|------|---------------------------|-------|-------------|
| 1    | Title & Pattern Intro     | 2 min | ★☆☆         |
| 2    | Guided Conversation       | 4 min | ★☆☆         |
| 3    | Memory Conversation       | 4 min | ★★☆         |
| 4    | Pattern Drill A           | 4 min | ★★☆         |
| 5    | Pattern Drill B           | 4 min | ★★☆         |
| 6    | Short Dialogues           | 4 min | ★★★         |
| 7    | Free Discussion           | 3 min | ★★★         |

## Sample Lesson: "Be Verb — Introductions & Occupations"

### PAGE 1 — Title & Pattern Intro

```
+-------------------------------------------------------------------------+
|                                                                         |
|  LESSON 1: NICE TO MEET YOU!                                            |
|  ----------------------------                                           |
|                                                                         |
|  Outcome:                                                               |
|  By the end of this lesson, you will be able to introduce yourself      |
|  and describe people using the verb "be."                               |
|                                                                         |
|                                                                         |
|  TODAY'S PATTERNS                                                       |
|  ================                                                       |
|                                                                         |
|  1. Identity (신원/정체)                                                   |
|                                                                         |
|     * I am a [student].          저는 [학생]입니다.                         |
|     * You are a [teacher].       당신은 [선생님]입니다.                      |
|     * He/She is a [doctor].      그/그녀는 [의사]입니다.                     |
|                                                                         |
|                                                                         |
|  2. Description (묘사)                                                    |
|                                                                         |
|     * I am [happy].              저는 [행복]합니다.                         |
|     * They are [tired].          그들은 [피곤]합니다.                        |
|     * She is [from Korea].       그녀는 [한국] 출신입니다.                    |
|                                                                         |
|                                                                         |
|  [Tutor: Read each pattern aloud. Student repeats after you.]           |
|                                                                         |
|                                                                  PAGE 1 |
+-------------------------------------------------------------------------+
```

### PAGE 2 — Guided Conversation

```
+-------------------------------------------------------------------------+
|                                                                         |
|  CONVERSATION: GUIDED READING                                           |
|  ----------------------------                                           |
|                                                                         |
|  Context: Min-ji meets Alex at a language exchange meetup.              |
|                                                                         |
|  [Tutor Note: Read together. Swap roles and read again.]               |
|                                                                         |
|                                                                         |
|  MIN-JI:  Hello! I AM Min-ji. I AM a student.                          |
|           Nice to meet you!                                             |
|                                                                         |
|  ALEX:    Hi, Min-ji! I AM Alex. I AM a teacher.                       |
|           Nice to meet you too!                                         |
|                                                                         |
|  MIN-JI:  IS THAT your friend over there?                              |
|                                                                         |
|  ALEX:    Yes! THAT IS Sarah. SHE IS from London.                      |
|           SHE IS very kind.                                             |
|                                                                         |
|  MIN-JI:  Great! And who ARE THOSE people?                              |
|                                                                         |
|  ALEX:    THEY ARE my classmates. THEY ARE really fun!                  |
|                                                                         |
|                                                                         |
|  [Target patterns are shown in CAPS for visibility]                     |
|                                                                         |
|                                                                         |
|                                                                  PAGE 2 |
+-------------------------------------------------------------------------+
```

### PAGE 4 — Pattern Drill A (Identity)

```
+-------------------------------------------------------------------------+
|                                                                         |
|  PATTERN DRILL: IDENTITY (신원/정체)                                      |
|  -----------------------------------                                    |
|                                                                         |
|  PATTERN: I am / You are / He is / She is / They are + [noun]          |
|                                                                         |
|                                                                         |
|  1. Read Together (같이 읽기):                                            |
|     * I am a student.                                                   |
|     * She is a doctor.                                                  |
|     * They are office workers.                                          |
|                                                                         |
|                                                                         |
|  2. Fill in the Blanks (빈칸 채우기):                                      |
|     * He _______ a teacher.                                             |
|     * We _______ students.                                              |
|     * _______ she a nurse?                                              |
|                                                                         |
|                                                                         |
|  3. Translate (영어로 번역해 보세요):                                        |
|                                                                         |
|     * 저는 요리사입니다.                                                    |
|       _____________________________________________________             |
|       (chef)                                                            |
|                                                                         |
|     * 그들은 경찰관입니다.                                                   |
|       _____________________________________________________             |
|       (police officers)                                                 |
|                                                                         |
|                                                                  PAGE 4 |
+-------------------------------------------------------------------------+
```

### PAGE 5 — Pattern Drill B (Description)

```
+-------------------------------------------------------------------------+
|                                                                         |
|  PATTERN DRILL: DESCRIPTION (묘사)                                       |
|  ---------------------------------                                      |
|                                                                         |
|  PATTERN: Subject + am/is/are + [adjective / from + place]             |
|                                                                         |
|                                                                         |
|  1. Read Together (같이 읽기):                                            |
|     * I am happy.                                                       |
|     * He is from Japan.                                                 |
|     * They are very tired.                                              |
|                                                                         |
|                                                                         |
|  2. Fill in the Blanks (빈칸 채우기):                                      |
|     * She _______ from Seoul.                                           |
|     * I _______ very excited.                                           |
|     * _______ you tired?                                                |
|                                                                         |
|                                                                         |
|  3. Translate (영어로 번역해 보세요):                                        |
|                                                                         |
|     * 우리는 한국에서 왔어요.                                                 |
|       _____________________________________________________             |
|       (Korea)                                                           |
|                                                                         |
|     * 그녀는 매우 친절합니다.                                                 |
|       _____________________________________________________             |
|       (very / kind)                                                     |
|                                                                         |
|                                                                  PAGE 5 |
+-------------------------------------------------------------------------+
```

### PAGE 7 — Free Discussion

```
+-------------------------------------------------------------------------+
|                                                                         |
|  FREE DISCUSSION                                                        |
|  ---------------                                                        |
|                                                                         |
|  [Tutor Note: Move from reading to free speaking. Use these             |
|  prompts. Help the student if they get stuck.]                          |
|                                                                         |
|                                                                         |
|  1. Introduce yourself to me! (자기소개를 해 보세요!)                        |
|     Tell me your name and your job (or what you want to be).            |
|                                                                         |
|     Sentence starters:                                                  |
|     * "I am ___."                                                       |
|     * "I am a ___."                                                     |
|                                                                         |
|                                                                         |
|  2. Describe someone in your family. (가족 중 한 명을 설명해 보세요.)          |
|     Tell me about one family member.                                    |
|                                                                         |
|     Sentence starters:                                                  |
|     * "He/She is ___."                                                  |
|     * "He/She is a ___."                                                |
|     * "He/She is from ___."                                             |
|                                                                         |
|                                                                         |

|     Use an adjective to describe yourself.                              |
|                                                                         |
|                                                                         |
|                                                                  PAGE 7 |
+-------------------------------------------------------------------------+
```
