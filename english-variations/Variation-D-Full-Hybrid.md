# Variation D: Full Hybrid

## Philosophy

This variation combines **all four approaches** into a single comprehensive structure.
It takes John's progressive drill framework, wraps it in Yeji's comfort-first tutor
scripts, layers in Scott's CLT communicative activities, and adds Andrew's visual
emphasis. The result is a longer, richer lesson that balances safety with challenge.

Best for: When you want the most complete lesson experience and have a full 25 minutes.

**Key Features:**
- Full tutor scripts on key pages (Yeji)
- Progressive pattern drills: read → fill → translate (John)
- 1 CLT activity with scaffolding (Scott)
- Visual cues and pictures where possible (Andrew)
- Korean translations throughout (all)
- "Mission Complete" wrap-up (Yeji)
- Clear student/tutor separation on every page
- 25 minutes total

---

## Lesson Flow (9 Pages)

| Page | Activity                     | Time  | Source Inspiration     |
|------|------------------------------|-------|------------------------|
| 1    | Welcome & Pattern Intro      | 2 min | Yeji + John            |
| 2    | Vocab Table                  | 2 min | Andrew + Scott         |
| 3    | Guided Conversation          | 3 min | John                   |
| 4    | Memory Conversation          | 3 min | John                   |
| 5    | Pattern Drill A              | 3 min | John                   |
| 6    | Pattern Drill B              | 3 min | John                   |
| 7    | Short Dialogues              | 3 min | John                   |
| 8    | CLT Activity (Roleplay)      | 4 min | Scott                  |
| 9    | Mission Complete             | 2 min | Yeji + Scott           |

## Sample Lesson: "Be Verb — Introductions & Occupations"

### PAGE 1 — Welcome & Pattern Intro

```
+-------------------------------------------------------------------------+
|                                                                         |
|  LESSON 1: NICE TO MEET YOU!                                            |
|  ----------------------------                                           |
|                                                                         |
|  ┌─── TUTOR SCRIPT ───────────────────────────────────────────┐         |
|  │  "안녕하세요! 오늘 재밌게 영어 공부 해봐요!                       │         |
|  │   간단한 영어만 사용할게요, 걱정 마세요!"                         │         |
|  │                                                             │         |
|  │  "Hello! Let's have fun learning English today!             │         |
|  │   We'll only use simple English — don't worry!"             │         |
|  │                                                             │         |
|  │  → Read patterns below. Student repeats after you.          │         |
|  └─────────────────────────────────────────────────────────────┘         |
|                                                                         |
|  Outcome:                                                               |
|  By the end of this lesson, you will be able to introduce               |
|  yourself and describe other people using the verb "be."                |
|                                                                         |
|                                                                         |
|  TODAY'S PATTERNS                                                       |
|  ================                                                       |
|                                                                         |
|  1. Identity (신원)                                                      |
|     I am a [student].          저는 [학생]입니다.                          |
|     She is a [doctor].         그녀는 [의사]입니다.                         |
|                                                                         |
|  2. Description (묘사)                                                   |

|                                                                         |
```


### PAGE 5 — Pattern Drill A (Identity)

```
+-------------------------------------------------------------------------+
|                                                                         |
|  PATTERN DRILL: IDENTITY (신원)                                          |

|                                                                         |
|                                                                         |
```

### PAGE 8 — CLT Roleplay Activity

```
+-------------------------------------------------------------------------+
|                                                                         |
|  REAL-LIFE PRACTICE: ROLEPLAY 실전 연습                                   |
|  ────────────────────────────────────                                   |
|  "The Welcome Party"                                                    |
|                                                                         |
|  SCENARIO:                                                              |
|  You are at a welcome party for new international students.             |
|  Meet the tutor (another student) and make a new friend!               |
|  상황: 외국인 학생 환영 파티에 왔습니다. 튜터(다른 학생)를 만나세요!              |
|                                                                         |
|                                                                         |
|  ┌─────────────────────────────────────────────────────────┐            |
|  │  YOUR GOAL: Make a new friend!                           │            |
|  │  목표: 새 친구를 사귀세요!                                    │            |
|  │                                                          │            |
|  │  RULES (규칙):                                            │            |
|  │  ✓ Introduce yourself (name + job/student)               │            |
|  │  ✓ Ask 2 questions using "Are you...?"                   │            |
|  │  ✓ Describe one thing about yourself ("I am...")         │            |
|  │  ✗ You cannot just say "Yes" or "No"                     │            |
|  │    (Use full sentences: "I am..." / "She is...")         │            |
|  └─────────────────────────────────────────────────────────┘            |
|                                                                         |
|                                                                         |
|  PHRASE BANK (이 표현을 사용하세요):                                        |
|  * "I am [name]. I am a [job]."                                         |
|  * "Are you from ___?"                                                  |
|  * "Are you a ___?"                                                     |
|  * "I am from ___."                                                     |
|  * "I am [feeling]."                                                    |
|                                                                         |
|  ┌─── TUTOR SCRIPT ────────────────────────────────────────┐            |
|  │  Start with: "Hi! Are you new here?"                     │            |
|  │  Use simple sentences. Stay in character.                │            |
|  │  If student is stuck, point to the phrase bank.          │            |
|  └──────────────────────────────────────────────────────────┘            |
|                                                                  PAGE 8 |
+-------------------------------------------------------------------------+
```
