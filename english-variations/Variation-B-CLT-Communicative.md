# Variation B: CLT Communicative

## Philosophy

This variation puts **Communicative Language Teaching** at the center. Inspired by
Scott's functional approach, every activity has a **real-world goal** the student must
achieve. Grammar is a tool, not the destination. Students learn by doing.

Best for: Students who are motivated by practical outcomes and real-life scenarios.

**Key Features:**
- CEFR-aligned functional goals (Scott)
- Every activity has a clear task/goal (Scott)
- Scaffolding with sentence starters & vocab support (Scott)
- Roleplay, info-gap, and task-based activities
- Korean translations as safety net
- 25 minutes total

---

## Lesson Flow (8 Pages)

| Page | Activity                     | Time  | Student Talking | Type       |
|------|------------------------------|-------|-----------------|------------|
| 1    | Title & Goal                 | 1 min | Low             | Setup      |
| 2    | Vocab & Phrases              | 3 min | Medium          | Input      |
| 3    | Pattern Introduction         | 3 min | Medium          | Input      |
| 4    | Guided Conversation          | 3 min | High            | Practice   |
| 5    | Info Gap Activity            | 5 min | Very High       | CLT        |
| 6    | Task-Based Activity          | 5 min | Very High       | CLT        |
| 7    | Roleplay with Constraint     | 4 min | Very High       | CLT        |
| 8    | Review & Goal Check          | 1 min | Low             | Wrap-up    |

## Sample Lesson: "Be Verb — Introductions & Occupations"

### PAGE 1 — Title & Goal

```
+-------------------------------------------------------------------------+
|                                                                         |
|  LESSON 1: NICE TO MEET YOU!                                            |
|  ----------------------------                                           |
|                                                                         |
|  Topic:    Introductions & Occupations                                  |
|  Level:    A1 Beginner                                                  |
|                                                                         |
|                                                                         |
|  ┌───────────────────────────────────────────────────────────┐          |
|  │  TODAY'S GOAL                                             │          |
|  │                                                           │          |
|  │  "I can introduce myself and ask about other people."     │          |
|  │                                                           │          |
|  │  오늘의 목표:                                                │          |
|  │  "나 자신을 소개하고 다른 사람에 대해 물어볼 수 있다."              │          |

### PAGE 3 — Pattern Introduction

```
+-------------------------------------------------------------------------+
|                                                                         |
|  GRAMMAR PATTERN                                                        |
|  ---------------                                                        |
|                                                                         |
|  We use "be" to say WHO we are, WHERE we're from, and HOW we feel.     |
|                                                                         |
|  ┌───────────────────────────────────────────────────────────┐          |
|  │  I        → am    a student / happy / from Korea          │          |
|  │  He/She   → is    a doctor / tired / from Japan           │          |
|  │  You/They → are   teachers / excited / from the USA       │          |
|  └───────────────────────────────────────────────────────────┘          |
|                                                                         |
|                                                                         |
|  FILL IN THE GAP (빈칸 채우기):                                           |
|                                                                         |
|  1. You _______ a very good teacher. (당신은 매우 좋은 선생님이에요.)        |
|  2. She _______ from Seoul. (그녀는 서울 출신이에요.)                       |
|  3. We _______ not tired. (우리는 피곤하지 않아요.)                          |
|  4. _______ he a doctor? (그는 의사인가요?)                                |
|                                                                         |
|                                                                         |
|  WORD ORDER (올바른 순서로 배열하세요):                                       |
|                                                                         |
|  1. excited / am / I / very         → ________________________          |
|  2. from / is / London / he         → ________________________          |
|  3. friends / are / you / my / ?    → ________________________          |
|                                                                         |
|                                                                  PAGE 3 |
+-------------------------------------------------------------------------+
```

+-------------------------------------------------------------------------+

|  ┌──────────┬───────────┬───────────┐                                   |
|  │  Name    │  Country  │  Job      │                                   |

|  └──────────┴───────────┴───────────┘                                   |

|  PHRASE BANK (이 표현을 사용하세요):                                        |

```

|                                                                         |
|                                                                         |
|  YOUR BOSS (Tutor) SAYS:                                                |
|  "We have two people who want to join our team. Look at their           |
|  profiles and tell me who is the best choice."                          |
|                                                                         |
|                                                                         |
|  CANDIDATE PROFILES:                                                    |
|  ┌────────────────────────────┬────────────────────────────┐            |
|  │  CANDIDATE A: PAUL         │  CANDIDATE B: JANE         │            |
|  ├────────────────────────────┼────────────────────────────┤            |
|  │  From: USA                 │  From: UK                  │            |
|  │  Job: Teacher              │  Job: Office worker        │            |

|  └────────────────────────────┴────────────────────────────┘            |
|                                                                         |
|                                                                         |

### PAGE 7 — Roleplay: "The First Day"

```
+-------------------------------------------------------------------------+
|                                                                         |
|  CLT ACTIVITY 3: ROLEPLAY WITH RULES                                    |
|  ------------------------------------                                   |
|  "The First Day at English Camp"                                        |
|                                                                         |
|  SCENARIO:                                                              |
|  It is the first day of an English camp. You meet another student       |
|  (the tutor). Your goal is to make a new friend!                        |
|  상황: 영어 캠프 첫날입니다. 다른 학생(튜터)을 만났습니다.                        |
|                                                                         |
|                                                                         |
|  YOUR GOAL: Make a new friend.    목표: 새 친구를 사귀세요.                  |
|                                                                         |
|                                                                         |
|  ┌─────────────────────────────────────────────────────────┐            |
|  │  RULES (규칙):                                           │            |
|  │                                                          │            |
|  │  ✓ You MUST ask 3 questions starting with "Are you...?"  │            |
|  │  ✓ You MUST mention your JOB and where you ARE FROM.     │            |
|  │  ✗ You CANNOT say the word "Yes."                        │            |
|  │    (Use "I am" or "That is correct" instead.)            │            |
|  └─────────────────────────────────────────────────────────┘            |
|                                                                         |
|                                                                         |
|  USEFUL PHRASES:                                                        |
|  * "Are you from ___?"                                                  |
|  * "Are you a ___?"                                                     |
|  * "I am from ___."                                                     |
|  * "I am a ___."                                                        |
|  * "That is correct!"                                                   |
|                                                                         |
|                                                                  PAGE 7 |
+-------------------------------------------------------------------------+
```
