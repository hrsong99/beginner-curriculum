# Beginner Japanese - Lesson 1

## Class Fundamentals

- **CEFR Level:** A1
- **Topic:** Nice to Meet You! (Self-Introductions)
- **Lesson Goal:** "I can introduce myself and say what I do."
- **Patterns:**
  1. 私は [Name] です。 (I am [Name].)
  2. 私は [Occupation] です。 (I am a [Occupation].)

## Page 5: Pattern Drill 1

```
+-------------------------------------------------------------------------+
|                                                                         |
|  PATTERN DRILL 1: SAYING YOUR NAME (名前を言う)                          |
|  -----------------------------------------------                        |
|                                                                         |
|  PATTERN: 私は [Name] です。                                              |
|                                                                         |
|                                                                         |
|  1. Read Together (一緒に読みましょう):                                    |
|     * 私は田中です。        (I am Tanaka.)                                |
|     * 私はキムです。        (I am Kim.)                                   |
|                                                                         |
|                                                                         |
|  2. Fill in the Blanks (空欄を埋めましょう):                               |
|     * _______ は マイク _______ 。                                       |
|     * _______ は ユナ _______ 。                                         |
|                                                                         |
|                                                                         |
|  3. Translate (日本語に翻訳してください):                                   |
|                                                                         |
|     * 저는 사토입니다.                                                     |
|       _____________________________________________________             |
|       (Sato / 佐藤)                                                     |
|                                                                         |
|     * 저는 박입니다.                                                       |
|       _____________________________________________________             |
|       (Park / パク)                                                      |
|                                                                         |
|                                                                         |
|  [Tutor Note: After drills, ask these bonus questions:]                  |
|  * お名前は? (What is your name?)                                        |
|  * 先生の名前は何ですか? (What is the teacher's name?)                     |
|                                                                         |
|                                                                  PAGE 5 |
+-------------------------------------------------------------------------+
```
