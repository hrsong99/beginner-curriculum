# Beginner Japanese - Lesson 2

## Class Fundamentals

- **CEFR Level:** A1
- **Topic:** What's That? (Identifying Objects)
- **Lesson Goal:** "I can identify and point out objects near me and far away."
- **Patterns:**
  1. これは [Object] です。 (This is [Object].)
  2. あれは [Object] です。 (That is [Object].)

## Page 3: Conversation - Guided Reading

```
+-------------------------------------------------------------------------+
|                                                                         |
|  CONVERSATION 1: GUIDED READING (会話: ガイド付き)                        |
|  ------------------------------------------------                       |
|                                                                         |
|  Context: A student (Kim) is visiting a Japanese friend's               |
|  (Sato) house for the first time.                                       |
|  場面: キムさんが佐藤さんの家に初めて来ました。                              |
|                                                                         |
|  [Tutor Note: Read together, then swap roles.]                          |
|                                                                         |
|                                                                         |
|  SATO: いらっしゃい! (Welcome!)                                           |
|                                                                         |
|  KIM:  おじゃまします。あ、これは何ですか?                                  |
|        (Excuse me for intruding. Ah, what is this?)                     |
|                                                                         |
|  SATO: これは日本のお茶です。どうぞ!                                       |
|        (This is Japanese tea. Here you go!)                             |
|                                                                         |
|  KIM:  ありがとうございます。あれは何ですか?                                |
|        (Thank you. What is that over there?)                            |
|                                                                         |
|  SATO: あれは猫です。名前はタマです。                                       |
|        (That is a cat. Its name is Tama.)                               |
|                                                                         |
|  KIM:  かわいい!                                                         |
|        (Cute!)                                                          |
|                                                                         |
```

```
|       (時計)                                                             |
|                                                                         |
```
