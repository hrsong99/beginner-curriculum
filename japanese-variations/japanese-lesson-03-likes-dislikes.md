# Beginner Japanese - Lesson 3

## Class Fundamentals

- **CEFR Level:** A1
- **Topic:** I Like It! (Expressing Likes & Dislikes)
- **Lesson Goal:** "I can say what I like and ask others what they like."
- **Patterns:**
  1. [Thing] が好きです。 (I like [Thing].)
  2. [Thing] が好きですか? (Do you like [Thing]?)

## Page 3: Conversation - Guided Reading

```
+-------------------------------------------------------------------------+
|                                                                         |
|  CONVERSATION 1: GUIDED READING (会話: ガイド付き)                        |
|  ------------------------------------------------                       |
|                                                                         |
|  Context: Two classmates are eating lunch together and talking           |
|  about what they like.                                                  |
|  場面: 二人のクラスメートが一緒にお昼を食べています。                          |
|                                                                         |
|  [Tutor Note: Read together, then swap roles.]                          |
|                                                                         |
|                                                                         |
|  A: ラーメンが好きですか?                                                  |
|     (Do you like ramen?)                                                |
|                                                                         |
|  B: はい、ラーメンが好きです! コーヒーが好きですか?                           |
|     (Yes, I like ramen! Do you like coffee?)                            |
|                                                                         |
|  A: いいえ、コーヒーが好きじゃないです。お茶が好きです。                       |
|     (No, I don't like coffee. I like tea.)                              |
|                                                                         |
|  B: そうですか。猫が好きですか?                                            |
|     (Is that so? Do you like cats?)                                     |
|                                                                         |
|  A: はい! 猫が大好きです!                                                  |
|     (Yes! I love cats!)                                                 |
|                                                                         |

## Page 9: Discussion - Find Your Match!

```
+-------------------------------------------------------------------------+
|                                                                         |
|  DISCUSSION: FIND YOUR MATCH! (合うものを見つけよう!)                       |
|  ---------------------------------------------------                    |
|                                                                         |
|  [Tutor Note: This is a mini information-gap activity.                  |
|  Both tutor and student take turns asking and answering.]               |
|                                                                         |
|                                                                         |
|  Goal: Find 2 things you BOTH like!                                     |
|  목표: 둘 다 좋아하는 것 2가지를 찾으세요!                                  |
|                                                                         |
|                                                                         |
|  CHECKLIST - Ask about each one!                                        |
|  +-------------------+----------+----------+                            |
|  | Item              | You      | Tutor    |                            |
|  +-------------------+----------+----------+                            |
|  | 寿司              |          |          |                            |
|  | ラーメン           |          |          |                            |
|  | コーヒー           |          |          |                            |
|  | 映画              |          |          |                            |
|  | 猫               |          |          |                            |
|  | 犬               |          |          |                            |
|  +-------------------+----------+----------+                            |
|                                                                         |
|  Sentence Starters:                                                     |
|  * _____ が好きですか?  (Do you like _____?)                              |
|  * はい、好きです。      (Yes, I like it.)                                |
|  * いいえ、好きじゃないです。 (No, I don't like it.)                       |
|                                                                         |
|  When done, tell the tutor:                                             |
|  「私たちは _____ と _____ が好きです!」                                    |
