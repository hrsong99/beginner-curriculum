# Variation D: Full Hybrid — Japanese (for English Speakers)

## Philosophy

Same structure as Variation D, adapted for **Japanese learners with English L1**.
Every Japanese line follows a clean three-line format: Japanese → Romaji → English.
English translations serve as the safety net throughout.

Best for: English-speaking absolute beginners learning Japanese in a 1:1 online class.

**Display Format:**
```
  Japanese text
  romaji reading
  (English meaning)
```

**Key Features:**
- Clean 3-line format: Japanese → Romaji → English
- Full tutor scripts (Yeji)
- Progressive pattern drills (John)
- 1 CLT-style roleplay with scaffolding (Scott)
- "Mission Complete" wrap-up (Yeji)
- 25 minutes total

---

## Lesson Flow (9 Pages)

| Page | Activity                     | Time  |
|------|------------------------------|-------|
| 1    | Welcome & Pattern Intro      | 2 min |
| 2    | Vocab Table                  | 2 min |
| 3    | Guided Conversation          | 3 min |
| 4    | Memory Conversation          | 3 min |
| 5    | Pattern Drill A              | 3 min |
| 6    | Pattern Drill B              | 3 min |
| 7    | Short Dialogues              | 3 min |
| 8    | CLT Activity (Roleplay)      | 4 min |
| 9    | Mission Complete             | 2 min |

## Sample Lesson: "desu / ja arimasen — Self-Intro & Occupations"

### PAGE 1 — Welcome & Pattern Intro

```
+-----------------------------------------------------------------------+
|                                                                       |
|  LESSON 1: はじめまして！                                               |
|            Hajimemashite!                                             |
|            (Nice to meet you!)                                        |
|  ────────────────────────                                             |
|                                                                       |
|  ┌─── TUTOR SCRIPT ─────────────────────────────────────────┐         |
|  │                                                           │         |
|  │  "Hello [Student Name]! Today we're going to learn        │         |
|  │  how to introduce yourself in Japanese. We'll only use    │         |
|  │  very simple Japanese — don't worry!"                     │         |
|  │                                                           │         |
|  │  → Read the patterns below aloud.                         │         |
|  │  → Student repeats after you.                             │         |
|  └───────────────────────────────────────────────────────────┘         |
|                                                                       |
|  Outcome:                                                             |
|  By the end of this lesson, you can introduce yourself and            |
|  describe people using "desu" and "ja arimasen."                      |
|                                                                       |
|                                                                       |
|  TODAY'S PATTERNS                                                     |
|  ================                                                     |
|                                                                       |
|  1. Saying what something IS:                                         |
|                                                                       |
|     私は [学生] です。                                                  |
|     Watashi wa [gakusei] desu.                                        |
|     (I am a [student].)                                               |
|                                                                       |
|                                                                       |
|  2. Saying what something IS NOT:                                     |
|                                                                       |
|     私は [先生] じゃありません。                                          |
|     Watashi wa [sensei] ja arimasen.                                  |
|     (I am not a [teacher].)                                           |
|                                                                       |
|                                                                PAGE 1 |
+-----------------------------------------------------------------------+
```

### PAGE 3 — Guided Conversation

```
+-----------------------------------------------------------------------+
|                                                                       |
|  CONVERSATION: GUIDED READING                                         |
|  ────────────────────────────                                         |
|                                                                       |
|  Context: Tom meets Yuki at a language exchange event.                |
|                                                                       |
|  [Tutor: Read together. Then swap roles and read again.]              |
|                                                                       |
|                                                                       |
|  TOM:    はじめまして！私はトムです。私は学生です。                         |
|          Hajimemashite! Watashi wa Tomu desu. Watashi wa gakusei desu.|
|          (Nice to meet you! I am Tom. I am a student.)                |
|                                                                       |
|  YUKI:   はじめまして！私はユキです。私は先生です。                         |
|          Hajimemashite! Watashi wa Yuki desu. Watashi wa sensei desu. |
|          (Nice to meet you! I am Yuki. I am a teacher.)               |
|                                                                       |
|  TOM:    よろしくお願いします！あの人は誰ですか？                           |
|          Yoroshiku onegaishimasu! Ano hito wa dare desu ka?           |
|          (Pleased to meet you! Who is that person?)                   |
|                                                                       |
|  YUKI:   あの人は田中さんです。田中さんは医者です。                         |
|          Ano hito wa Tanaka-san desu. Tanaka-san wa isha desu.        |
|          (That person is Mr. Tanaka. Mr. Tanaka is a doctor.)         |
|                                                                       |
|  YUKI:   田中さんはとても親切です。                                       |
|          Tanaka-san wa totemo shinsetsu desu.                         |
|          (Mr. Tanaka is very kind.)                                   |
|                                                                       |
|                                                                PAGE 3 |
+-----------------------------------------------------------------------+
```

### PAGE 8 — CLT Roleplay Activity

```
+-----------------------------------------------------------------------+
|                                                                       |
|  REAL-LIFE PRACTICE: ROLEPLAY                                         |
|  ────────────────────────────                                         |
|  "First Day at Japanese Class"                                        |
|                                                                       |
|  SCENARIO:                                                            |
|  You just arrived at a Japanese class abroad. You meet another        |
|  student (the tutor). Introduce yourself and get to know them!        |
|                                                                       |
|                                                                       |
|  ┌─────────────────────────────────────────────────────────┐          |
|  │  YOUR GOAL: Introduce yourself & learn about the tutor. │          |
|  │                                                         │          |
|  │  RULES:                                                 │          |
|  │  ✓ Say your name and job/role using "desu"              │          |
|  │  ✓ Ask at least 1 question using "desu ka?"             │          |
|  │  ✓ Say 1 thing that is NOT true using "ja arimasen"     │          |
|  │  ✗ No answering with just "hai" or "iie" — use full     │          |
|  │    sentences!                                           │          |
|  └─────────────────────────────────────────────────────────┘          |
|                                                                       |
|                                                                       |
|  PHRASE BANK:                                                         |
|                                                                       |
|  * 私は___です。                                                       |
|    Watashi wa ___ desu.              (I am ___.)                      |
|                                                                       |
|  * ___は___ですか？                                                     |
|    ___ wa ___ desu ka?               (Is ___ a ___?)                  |
|                                                                       |
|  * 私は___じゃありません。                                                |
|    Watashi wa ___ ja arimasen.       (I am not ___.)                  |
|                                                                       |
|  * よろしくお願いします！                                                  |
|    Yoroshiku onegaishimasu!          (Pleased to meet you!)           |
|                                                                       |
|                                                                       |
|  ┌─── TUTOR SCRIPT ──────────────────────────────────────┐            |
|  │  Start with:                                           │            |
|  │  "はじめまして！私は___です。"                              │            |
|  │  Hajimemashite! Watashi wa ___ desu.                   │            |
|  │                                                        │            |
|  │  Keep sentences simple. If student is stuck,           │            |
|  │  point to the phrase bank.                             │            |
|  └────────────────────────────────────────────────────────┘            |
|                                                                       |
|                                                                PAGE 8 |
+-----------------------------------------------------------------------+
```
