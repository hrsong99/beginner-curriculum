# Variation E: Simplified Hybrid

## Philosophy

This variation is a **streamlined version of the Full Hybrid (D)**, cutting the vocab page
and simplifying tutor-facing elements to let the lesson flow faster. It keeps the progressive
drill framework and CLT roleplay but trims overhead so tutors spend less time reading
instructions and more time interacting with the student.

Best for: When you want a complete lesson experience in under 20 minutes with less tutor prep.

**Key Features:**
- Clean pattern intro without tutor script (Variation A style)
- Progressive pattern drills: read → fill → translate (John)
- 1 CLT activity with bilingual goal boxes (Scott)
- English translations throughout (all)
- "Mission Complete" wrap-up with big checkboxes (Yeji)
- Clear student/tutor separation on every page
- ~20 minutes total

---

## Lesson Flow (9 Pages)

| Page | Activity                     | Time  | Source Inspiration     |
|------|------------------------------|-------|------------------------|
| 1    | Title & Pattern Intro        | 2 min | Variation A            |
| 2    | Guided Conversation          | 3 min | John                   |

| 4    | Pattern Drill A              | 3 min | John                   |
| 5    | Pattern Drill B              | 3 min | John                   |
| 6    | Short Dialogues              | 3 min | John                   |
| 7    | CLT Activity (Roleplay)      | 4 min | Scott                  |
| 8    | Free Discussion              | 3 min | Variation A            |

## Sample Lesson: "これ・それ・あれ — Things Around You"

### PAGE 1 — Title & Pattern Intro

```
+-------------------------------------------------------------------------+
|                                                                         |
|  LESSON 2: これはなんですか？                                              |
|  ---------------------------                                            |
|                                                                         |
|  ┌─── OUTCOME / 目標 ──────────────────────────────────────────┐         |
|  │  You can identify things and talk about who they belong to  │         |
|  │  物を指して、誰の物か話せるようになります。                       │         |
|  └─────────────────────────────────────────────────────────────┘         |
|                                                                         |
|                                                                         |
|  TODAY'S PATTERNS                                                       |
|  ================                                                       |
|                                                                         |
|  1. Demonstratives (指示語)                                               |
|                                                                         |
|     * これは[本]です。                This is a [book].                    |
|     * それは[ペン]です。              That is a [pen].                     |
|     * あれは[かばん]です。            That over there is a [bag].          |
|                                                                         |
|                                                                         |
|  2. Possession (所有)                                                     |
|                                                                         |
|     * これは[私]の[本]です。          This is [my] [book].                 |
|     * それは[先生]の[ペン]です。      That is [the teacher]'s [pen].       |
|     * あれは[ゆき]の[かばん]です。    That is [Yuki]'s [bag].              |
|                                                                         |
|                                                                         |
|                                                                  PAGE 1 |
+-------------------------------------------------------------------------+
```

### PAGE 4 — Pattern Drill A (Demonstratives)

```
+-------------------------------------------------------------------------+
|                                                                         |
|  PATTERN DRILL: DEMONSTRATIVES (指示語)                                  |
|  --------------------------------------                                 |
|                                                                         |
|  PATTERN: これ/それ/あれは [noun]です。                                   |
|                                                                         |
|                                                                         |
|  1. Read Together:                                                      |
|     * これは本です。                    This is a book.                   |
|     * それはペンです。                  That is a pen.                    |
|     * あれは時計です。                  That over there is a clock.       |
|                                                                         |
|                                                                         |
|  2. Fill in the Blanks:                                                 |
|     * （　　）は本です。                This is a book.                   |
|     * （　　）はペンです。              That (near you) is a pen.         |
|     * （　　）は時計です。              That (far away) is a clock.       |
|                                                                         |
|                                                                         |
|  3. Translate (Translate to Japanese):                                  |
|                                                                         |
|     * This is a phone.                                                  |
|       _____________________________________________________             |
|       (電話 / でんわ)                                                    |
|                                                                         |
|     * That over there is a school.                                      |
|       _____________________________________________________             |
|       (学校 / がっこう)                                                   |
|                                                                         |
|                                                                         |
|  4. Make Your Own:                                                      |
|     Point at something and make a sentence!                             |
|                                                                         |
|     _____________________________________________________               |
|                                                                         |
|                                                                  PAGE 4 |
+-------------------------------------------------------------------------+
```

### PAGE 7 — CLT Roleplay Activity

```
+-------------------------------------------------------------------------+
|                                                                         |
|  REAL-LIFE PRACTICE: ROLEPLAY                                           |
|  ────────────────────────────                                           |
|  "The Lost & Found"                                                     |
|                                                                         |
|  SCENARIO:                                                              |
|  You are at the school lost & found. The tutor works there.             |
|  Ask about items and find your things!                                  |
|  状況: 学校の落とし物コーナーにいます。チューターが担当者です。           |
|  アイテムについて聞いて、自分の物を見つけましょう！                       |
|                                                                         |
|                                                                         |
|  ┌─── チューター用 ───────────────────────────────────────┐            |
|  │                                                         │            |
|  │  目標: 学生が自然な会話をできるようにサポートする。        │            |
|  │                                                         │            |
|  │  ルール:                                                 │            |
|  │  ✓ "いらっしゃいませ！何をお探しですか？" から始める      │            |
|  │  ✓ 簡単な文を使う。役になりきる。                         │            |
|  │  ✗ 会話中に文法を直さない。                               │            |
|  │                                                         │            |
|  │  フレーズ:                                               │            |
|  │  * "これですか？"                                        │            |
|  │  * "それは___さんのですか？"                              │            |
|  │                                                         │            |
|  └─────────────────────────────────────────────────────────┘            |
|                                                                         |
|                                                                         |
|  ┌─── FOR STUDENT ────────────────────────────────────────┐            |
|  │                                                         │            |
|  │  GOAL: Find your things!                                │            |
|  │                                                         │            |
|  │  RULES:                                                 │            |
|  │  ✓ Describe items using これ/それ/あれ                    │            |
|  │  ✓ Use の to say whose things they are                   │            |
|  │  ✗ Don't just say "はい" / "いいえ"!                     │            |
|  │                                                         │            |
|  │  USE THESE PHRASES:                                     │            |
|  │  * "あれは私のかばんです。"                               │            |
|  │  * "これは___の___です。"                                 │            |
|  │                                                         │            |
|  └─────────────────────────────────────────────────────────┘            |
|                                                                         |
|                                                                  PAGE 7 |
+-------------------------------------------------------------------------+
```
